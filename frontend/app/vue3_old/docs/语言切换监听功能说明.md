# 语言切换监听功能使用说明

## 📋 功能概述

当用户切换语言时，所有调用了 `list` 和 `get` API 的页面会自动监听语言切换事件，并重新请求 API 获取相应语言的文档。

## 🏗️ 架构设计

### 核心组件

1. **useLanguageChangeStore** (`src/stores/modules/app/language-change.state.ts`)
   - 管理语言切换监听的核心 Store
   - 提供回调注册、触发等机制

2. **useLanguageChangeEffect** (`src/hooks/useLanguageChangeEffect.ts`)
   - 简化使用的 Composable
   - 自动处理组件生命周期

3. **useLanguageChangeListener** (`src/hooks/useLanguageChangeEffect.ts`)
   - 页面级别的 Hook
   - 更简洁的 API

### 工作原理

```
用户切换语言
    ↓
i18n locale 改变
    ↓
App.vue 中的监听器检测到变化
    ↓
triggerLanguageChange() 触发所有回调
    ↓
各个页面的回调函数执行
    ↓
重新调用 API (自动使用新语言)
    ↓
数据更新，UI 自动刷新
```

## 🚀 使用方式

### 方式 1: 在页面中使用 (推荐)

```typescript
<script setup lang="ts">
import {useLanguageChangeEffect} from '@/hooks';
import {usePostStore, useCategoryStore} from '@/stores';

const postStore = usePostStore();
const categoryStore = useCategoryStore();

async function loadData() {
  await Promise.all([
    postStore.listPost({page: 1, pageSize: 10}),
    categoryStore.listCategory({page: 1, pageSize: 10}),
  ]);
}

// 监听语言切换，自动重新加载数据
useLanguageChangeEffect(loadData, {
  immediate: true,      // 是否立即执行一次
  autoCleanup: true,    // 是否自动清理
});
</script>
```

### 方式 2: 在文章详情页使用

```typescript
<script setup lang="ts">
import {ref, onMounted} from 'vue';

  loadPostDetail();

```

## 📝 完整示例：首页

```typescript
<script setup lang="ts">
import {ref, onMounted, onUnmounted} from 'vue';
import {useLanguageChangeEffect} from '@/hooks';
import {usePostStore, useCategoryStore, useTagStore} from '@/stores';

const postStore = usePostStore();
const categoryStore = useCategoryStore();
const tagStore = useTagStore();

const posts = ref([]);
const categories = ref([]);
const tags = ref([]);
const loading = ref(false);

// 加载数据函数
async function loadAllData() {
  loading.value = true;
  try {
    await Promise.all([
      postStore.listPost({page: 1, pageSize: 10}),
      categoryStore.listCategory({page: 1, pageSize: 10}),
      tagStore.listTag({page: 1, pageSize: 10}),
    ]);
  } finally {
    loading.value = false;
  }
}

// 监听语言切换
useLanguageChangeEffect(loadAllData, {
  immediate: false,     // onMounted 已加载，不需要立即执行
  autoCleanup: true,    // 自动清理
});

onMounted(async () => {
  await loadAllData();
});
</script>
```

## ⚙️ API 参考

### useLanguageChangeStore

```typescript
interface UseLanguageChangeStore {
  // 当前语言
  currentLanguage: string;
  
  // 是否正在切换
  isSwitching: boolean;
  
  // 注册语言切换回调
  watchLanguageChange(
    callback: () => void | Promise<void>
  ): () => void;  // 返回取消订阅函数
  
  // 手动触发语言切换
  triggerLanguageChange(): Promise<void>;
  
  // 初始化语言监听器
  initializeLanguageWatcher(): void;
  
  // 获取当前语言
  getCurrentLanguage(): string;
  
  // 清除所有回调
  clearAllCallbacks(): void;
}
```

### useLanguageChangeEffect

```typescript
function useLanguageChangeEffect(
  loadData: () => void | Promise<void>,
  options?: {
    // 是否立即执行一次 (默认 true)
    immediate?: boolean;
    
    // 是否在组件卸载时自动取消订阅 (默认 true)
    autoCleanup?: boolean;
  }
): () => void;  // 返回取消订阅函数
```

### useLanguageChangeListener

```typescript

```

## 🎯 注意事项

### 1. Store 自动获取当前语言

所有 store 的 `list` 和 `get` 方法都会自动使用 `currentLocaleLanguageCode()`:

```typescript
// post.state.ts
async function listPost(...) {
  const locale = currentLocaleLanguageCode(); // ✅ 自动获取
  formValues = {...formValues, locale};
  // ...
}
```

**无需手动传递 languageCode!**

### 2. 回调函数应该是幂等的

```typescript
// ✅ 好的做法
async function loadData() {
  await postStore.listPost({page: 1, pageSize: 10});
}

// ❌ 避免副作用
async function loadData() {
  counter++; // 不要累积状态
}
```

### 3. 错误处理

```typescript
// ✅ 推荐：在回调中处理错误
async function loadData() {
  try {
    await postStore.listPost({page: 1, pageSize: 10});
  } catch (error) {
    console.error('Load data failed:', error);
    // 可以显示错误提示
  }
}
```

### 4. 性能优化

```typescript
// ✅ 推荐：并行加载
async function loadData() {
  await Promise.all([
    postStore.listPost(),
    categoryStore.listCategory(),
    tagStore.listTag(),
  ]);
}

// ❌ 避免：串行加载
async function loadData() {
  await postStore.listPost();
  await categoryStore.listCategory();
  await tagStore.listTag();
}
```

### 5. 内存管理

```typescript
// ✅ autoCleanup: true 时自动清理
useLanguageChangeEffect(loadData, {
  autoCleanup: true
});

// ✅ 手动订阅时记得清理
const unsubscribe = languageChangeStore.watchLanguageChange(loadData);
onUnmounted(() => unsubscribe());

// ❌ 避免内存泄漏
languageChangeStore.watchLanguageChange(loadData);
// 没有清理
```

## 🐛 常见问题

### Q: 为什么语言切换了但数据没更新？

A: 检查是否正确使用了 `useLanguageChangeEffect`:

```typescript
// ✅ 正确
useLanguageChangeEffect(loadData);

// ❌ 错误：没有注册回调
loadData();
```

### Q: 需要手动调用 triggerLanguageChange 吗？

A: **不需要!** 系统会自动检测 i18n locale 的变化并触发。

### Q: 可以在多个地方注册回调吗？

A: **可以!** 每个页面/组件都可以独立注册回调。

### Q: 会影响性能吗？

A: 不会。回调只在语言切换时执行，且使用 Promise.all 并行加载。
